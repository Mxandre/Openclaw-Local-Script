# Openclaw-Local-Script
Personal collection of local Openclaw script, continuously updated to customize and enhance your Smart Agent Experience.  Share, experiment and personnalize your tools safely.

## University Email Summary script 
This script is designed for the university whose email serveur doesn't support Application password in the IMAPS serve.

Instead of exposing your email credential to the bots, the script runs locally and return a json format of the inbox. 

The Openclaw robot can then read the output and send a daily summary.
